# wallet

## Project setup
```
npm install
```

### Setup env variables

Create next files:
* .env.local
* .env.production.local
* .env.staging.local

Copy-paste all keys from .env file to this files.

Ask your teammate for keys values.

Check [official docs](https://cli.vuejs.org/guide/mode-and-env.html)
for more info.
